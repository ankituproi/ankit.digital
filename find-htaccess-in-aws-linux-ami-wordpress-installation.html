Find .htaccess in AWS Linux AMI wordpress installation

If you are new to AWS and is in trouble finding .htaccess file for your wordpress installation, then you are at the very apt place. You might have your wordpress installation using bitnami image on aws, or Linux AMI / Linux 2 AMI or Ubuntu or any other.

Like other shared hosting, Amazon VPC, does not have .htaccess file with the name in server. Rather you will be able to find it here : /etc/httpd/httpd.conf

You can carryout anything from here which you want to do through .htaccess file. Various operation can be done through httpd.conf file similar to that of .htaccess. Few examples :

  • Defining File paths
  • Redirection
  • Canonical rules
  • blocking / allowing ip's
  • blocking / allowing certain bots.
  • etc
